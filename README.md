# Claude Swap

AI-Powered Decentralized Swap Intelligence Platform for Solana

## üöÄ Quick Start

### Prerequisites

- Node.js 20.x or higher
- npm or pnpm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Claude-Swap/ClaudeSwap.git
cd ClaudeSwap
```

2. Install dependencies:
```bash
npm install
```

or with pnpm:
```bash
pnpm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
